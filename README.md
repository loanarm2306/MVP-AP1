# MVP-AP1

Repositorio de la Actividad Práctica I de la asignatura Metodologías de Gestión y Diseño de Proyectos Big Data.

El proyecto aplica CRISP-DM, Scrum, DataOps y MLOps sobre un caso de predicción de mora en créditos de una entidad financiera. Se parte de un MVP inicial desarrollado en notebooks y se incorporan prácticas de versionado, automatización de validaciones, scripts de transformación y pipelines reproducibles.

## Estructura inicial

- `notebooks/`: notebooks originales del MVP.
- `data/raw/`: datasets originales del proyecto.
- `src/`: scripts Python que se irán generando.
- `tests/`: pruebas automatizadas de calidad de datos.
- `reports/figures/`: gráficos y salidas generadas.
- `docs/`: documentación complementaria.
